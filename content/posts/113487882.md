  --unshallow无反应的解决方法

cd到/usr/local/Homebrew/Library/Taps/homebrew/homebrew-core目录下，执行命令：

```bash
git fetch 
```
成功之后再执行
```bash
git fetch --unshallow
```
对于/usr/local/Homebrew/Library/Taps/homebrew/homebrew-cask也是同理
