```bash
configure terminal
no access-list 100
access-list 100 permit ip any  host 139.155.65.168 
access-list 100 permit ip any  host 1.14.109.79
access-list 100 permit ip any  host 1.14.108.193 
access-list 100 permit ip 10.30.17.0 0.0.0.255 10.30.17.0 0.0.0.255 
access-list 100 deny ip any any

Interface range GigabitEthernet 0/2-24 #批量配置编号为2-24的23个端口，需要注意的是不能用全局ACL，也不能包含连接另一个交换机的端口，在这里是GigabitEthernet0/1，不然不起作用。
ip access-group 100 in
end
write

```
